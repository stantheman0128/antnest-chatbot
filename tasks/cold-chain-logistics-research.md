# 台灣冷藏/冷凍物流服務深度研究報告

> 研究日期：2026-03-23
> 適用對象：自建 POS / 電商網站串接冷鏈物流

---

## 一、各物流服務商詳細比較

### 1. 黑貓宅急便（統一速達）— 市佔率最高的低溫宅配

**溫層**：冷藏 0~7°C / 冷凍 -15°C 以下

**本島運費（牌價）**：

| 尺寸（長+寬+高） | 低溫運費 | 離島運費 |
| ---------------- | -------- | -------- |
| 60cm 以下        | 160 元   | 260 元   |
| 61~90cm          | 225 元   | 340 元   |
| 91~120cm         | 290 元   | 400 元   |

- 材積上限：120cm（長+寬+高），重量上限 20kg
- 繁盛期（除夕、中秋節前 1~7 日）加收 10 元服務費
- 配送時間：本島 D+1~D+2（翌日配），離島 D+3~D+5
- **冷藏與冷凍同價**，不分溫層

**月結/合約價**：

- 黑貓提供企業契約客戶專區，可進行線上出貨、查詢
- 月出貨量達一定門檻可談合約折扣（一般低溫合約價約 120~150 元/60cm 級距，視量而定）
- 需向當地營業所申請，提供營業登記資料

**API 串接**：

- 黑貓有提供自有 API，可匯入地址後產生託運單及託運編號
- 申請方式：向黑貓申請 API 串接需求，取得授權碼後設定
- 也可透過**綠界科技 (ECPay)** 的物流整合 API 串接黑貓宅配（含低溫）

**優點**：品質最穩定、溫控最佳、覆蓋率最高
**缺點**：牌價最貴、合約門檻較高

---

### 2. 台灣宅配通（大嘴鳥 Pelican Express / e-can）

**溫層**：冷藏 / 冷凍

**本島運費（牌價）**：

| 尺寸（長+寬+高） | 低溫運費 | 離島運費 |
| ---------------- | -------- | -------- |
| 60cm 以下        | 155 元   | 260 元   |
| 61~90cm          | 220 元   | 340 元   |
| 91~120cm         | 285 元   | 400 元   |

- 材積上限：120cm，單邊長 ≤ 75cm，重量上限 20kg
- 離島：單邊長 ≤ 50cm，重量上限 15kg
- 當日低溫快配（限北北基）：170~320 元
- 配送時間：D+1 或 D+2

**月結/合約價**：可談月結合約

**API 串接**：

- 可透過綠界科技 API 串接
- 部分電商平台（SHOPLINE、CYBERBIZ、meepShop）已整合

**優點**：價格比黑貓略便宜約 5~10 元/件
**缺點**：物流品質不如黑貓穩定，精緻商品（甜點蛋糕）業者較不愛用

---

### 3. 新竹物流

**溫層**：冷藏 0~7°C / 冷凍 -18°C

**本島運費（牌價）**：

| 尺寸（長+寬+高） | 低溫運費 |
| ---------------- | -------- |
| 60cm 以下（S60） | 150 元   |
| 61~90cm（S90）   | 210 元   |
| 91~120cm（S120） | 270 元   |

- 材積上限：120cm，重量上限 20kg
- 全程 GPS 監控、溫度控管、即時貨態追蹤
- 僅限本島配送

**月結/合約價**：

- 支援月結，但物流品質與黑貓有差距

**API 串接**：

- 有提供 API 串接
- 可透過鼎新 A1 等 ERP 系統整合

**優點**：價格在三大宅配中最便宜
**缺點**：低溫品質穩定性次於黑貓，無離島服務

---

### 4. 嘉里大榮物流 (Kerry TJ Logistics)

**溫層**：全程 7°C ~ -25°C 低溫環境管理

**運費**：

- 不公開牌價表，採「接費 + 本運費 + 送費 + 特殊服務費 + 聯運費」結構
- 依距離、材積、重量取大者計價
- 需聯繫業務專員報價

**服務特色**：

- 全台 19 處低溫營業據點 + 3 大轉運中心（含澎湖）
- 以 B2B 為主、B2C 為輔
- 適合需要長期低溫保存的商品（藥品、化妝品等）

**API 串接**：

- 有提供 API 串接功能
- 可透過鼎新 A1 等系統整合

**優點**：專業冷鏈管理、涵蓋澎湖
**缺點**：偏向 B2B 大宗、個人/小型電商門檻高、價格不透明

---

### 5. 中華郵政（郵局）

**結論：不提供低溫宅配服務**

- 郵局「便利包」僅限常溫、1kg 以內物品
- 曾有「優鮮配」冷凍冷藏服務，但因連年虧損於 2008 年後停辦
- 綠界 API 串接中華郵政，但**不提供低溫服務**

## 二、超商冷凍店到店服務

### 6. 7-ELEVEN 冷凍交貨便（店到店）

**溫層**：冷凍（-18°C 以下）

**運費**：

| 服務類型        | 優惠價 | 原價   |
| --------------- | ------ | ------ |
| 純取貨（C2C）   | 129 元 | 200 元 |
| 取貨付款（C2C） | 134 元 | 200 元 |

**離島**：原價 335 元

**專用紙箱**：

| 紙箱規格 | 尺寸       | 重量上限 | 紙箱費 |
| -------- | ---------- | -------- | ------ |
| S60      | 27×20×13cm | 10kg     | 12 元  |
| S80      | 35×25×20cm | 10kg     | 20 元  |
| S90      | 35×30×25cm | 10kg     | 20 元  |

- 三邊合計 ≤ 105cm，單邊 ≤ 45cm，重量 ≤ 20kg
- 配送時間：約 3~4 天送達指定門市
- 取件期限：到店 4 天內取件，逾期退回收 180 元逆物流費 + 15 元轉運費
- 需包裝為冷凍狀態（預冷 12 小時以上）

**重要**：僅提供冷凍，不提供冷藏店到店

---

### 7. 7-ELEVEN 賣貨便（店到宅）

**溫層**：冷藏 + 冷凍（由黑貓宅急便配送）

**運費**：

| 服務類型             | 優惠價 | 原價   |
| -------------------- | ------ | ------ |
| 常溫店到宅           | 180 元 | 210 元 |
| 冷凍店到宅           | 190 元 | 225 元 |
| 冷藏店到宅（新服務） | 190 元 | 225 元 |

- 常溫：三邊合計 ≤ 150cm，≤ 20kg
- 冷凍/冷藏：三邊合計 ≤ 90cm，≤ 20kg
- 配送時間：寄件後 2~3 個工作日送達
- 上架免費，買賣雙方只付物流費

### 8. 全家便利商店 冷凍店到店

**溫層**：冷凍（-18°C 以下）

**運費**：

| 服務類型        | 優惠價 | 原價       |
| --------------- | ------ | ---------- |
| 一般寄件（C2C） | 99 元  | 168~310 元 |
| 取貨付款        | 114 元 | 183~325 元 |

**紙箱規格**：

| 紙箱規格 | 尺寸       | 重量上限 | 紙箱費 |
| -------- | ---------- | -------- | ------ |
| S60      | 28×20×12cm | 5kg      | 12 元  |

- 自 2025/06/09 起，取消需使用全家冷凍寄件專用紙箱的限制

**全家好賣+平台**：提供社群賣家接單平台，整合金流、物流、資訊流

**重要**：僅提供冷凍，不提供冷藏店到店

---

### 9. 萊爾富 / OK 超商

- **萊爾富**：配合宅配通、新竹物流，有低溫宅配代收
- **OK 超商**：配合黑貓宅急便，有低溫宅配（含冷藏、冷凍）；低溫物品到店留存 5 天
- 兩者的冷凍店到店服務規模較小，門市冷凍櫃容量有限

## 三、電商平台物流 / 第三方平台

### 10. 蝦皮物流

- 蝦皮目前主力在常溫物流（超商取貨、蝦皮店到店）
- **無自有冷鏈物流服務**
- 平台上的冷凍食品賣家多自行委託黑貓/宅配通低溫宅配

### 11. PChome / momo

- PChome 24h、momo 購物有提供低溫配送，但僅限平台內賣家
- 無法串接外部自建電商網站
- 主要委託黑貓、宅配通進行配送

### 12. Lalamove 即時低溫配送

- 提供 1.75 噸、3.49 噸冷凍/冷藏車即時配送
- 服務區域：基隆、雙北、桃園、新竹（部分）、台中、台南、高雄
- 運費結構：起始價 + 每公里加價（即時動態報價）
- 適合：當日急送、活動期間大量出貨
- 有 EasyStore 等電商平台整合
- **不適合**日常小件冷凍宅配（成本高）

### 13. CYBERBIZ 電商平台

- 台灣唯一整合品牌官網 + 雲端 POS + 智能倉儲的平台
- **有自己的多溫層物流倉儲系統**
- 提供快速出貨、精準效期、即時貨態追蹤
- 適合需要大型倉儲及完善冷鏈的中大型商家
- 提供「CYBERBIZ NOW!」機車外送服務

### 14. 逢泰企業

- 台灣知名第三方低溫物流（3PL）
- 從事低溫食品倉儲、理貨、分裝、代理、配送
- 台灣低溫倉儲單一園區前 3 名
- 適合有大量冷鏈倉儲需求的企業

## 四、API 串接可能性總整理

### 第三方整合平台

#### 綠界科技 ECPay（最推薦的整合方案）

| 服務類型            | 支援物流商             | 低溫支援                  |
| ------------------- | ---------------------- | ------------------------- |
| 超商取貨（C2C/B2C） | 全家、7-11、萊爾富、OK | 有（7-11/全家冷凍店到店） |
| 宅配                | 黑貓宅急便             | **有低溫（冷藏/冷凍）**   |
| 宅配                | 中華郵政               | **無低溫**                |

- **API 格式**：Form-data，MD5 加密
- **SDK**：Python、.NET、PHP 等多語言
- **功能**：建立正逆物流訂單、電子地圖、託運單列印、貨態查詢
- **開發者文件**：https://developers.ecpay.com.tw/
- **串接難度**：中等，文件完整，有 AI Skill 套件可輔助
- **費用**：需申請特約商店，有年費及手續費

#### PayNow（奧丁丁）

- 支援全家冷凍大宗 B2C 物流
- 提供物流流程、超商電子地圖、物流單管理 API
- 串接文件：https://owlting.github.io/paynow-guideline/
- 適合已使用 PayNow 金流的商家

#### 藍新金流 NewebPay

- 主要提供**金流**服務
- 物流服務以常溫超商取貨為主
- **目前未明確支援冷鏈物流 API**
- 如需冷鏈，需另行串接物流商

#### ezShip 便利配

- 支援全家、萊爾富、OK 店到店
- 以常溫為主
- 商務會員年費 1,200 元

## 六、小型電商（月出貨 100~500 件）最佳方案建議

### 方案 A：最便宜 — 全家冷凍店到店（適合小件商品）

- **運費 99 元/件**（優惠價），取貨付款 114 元
- 限冷凍、5~10kg 以內
- 透過全家好賣+平台或綠界 API 串接
- 月出貨 100 件成本約 **9,900~11,400 元**
- 缺點：僅冷凍不冷藏、尺寸受限、依賴門市冷凍櫃空間

### 方案 B：性價比優 — 7-11 冷凍交貨便 + 賣貨便

- 店到店 129 元，店到宅 190 元
- 賣貨便可同時使用冷藏/冷凍店到宅
- 月出貨 100 件成本約 **12,900~19,000 元**

### 方案 C：品質最佳 — 黑貓低溫宅配（合約價）

- 牌價 160 元起，合約價可談到 **120~140 元**（60cm 級距）
- 月出貨 300 件以上較有議價空間
- 透過綠界 API 或黑貓自有 API 串接
- 月出貨 300 件成本約 **36,000~42,000 元**
- 品質最穩定、溫控最好、消費者信任度高

### 方案 D：自建官網推薦整合路線

1. **金流**：綠界 ECPay 或藍新 NewebPay
2. **常溫物流**：綠界 API 串接超商取貨（全家/7-11/萊爾富）
3. **冷凍物流**：
   - 小件 → 綠界 API 串接 7-11/全家冷凍店到店（99~134 元）
   - 大件 → 綠界 API 串接黑貓低溫宅配（160 元起）
   - 或直接串接黑貓 API 取得合約價
4. **冷藏物流**：
   - 7-11 賣貨便店到宅冷藏（190 元）
   - 黑貓低溫宅配冷藏（160 元起）

## 七、關鍵發現與結論

### 回答核心問題

**Q: 有沒有像 7-11 常溫 65 元那樣便宜的冷藏/冷凍物流？**

A: 沒有。冷鏈物流因需全程溫控，成本結構完全不同。目前最便宜的選項：

- **全家冷凍店到店 99 元** — 這是目前台灣最低價的冷凍物流選項
- **7-11 冷凍交貨便 129 元** — 第二便宜
- 宅配到府最低 **150 元**（新竹物流）

冷藏與常溫的價差約 **2~3 倍**，這是產業結構性成本。

### 超商冷藏取貨？

**目前超商店到店僅提供「冷凍」，不提供「冷藏」**。7-11 和全家的冷凍店到店都是 -18°C 冷凍溫層，因為超商門市冷凍櫃較容易統一管理，冷藏（0~7°C）的溫控較難在超商環境維持。

唯一的冷藏選項是 **7-11 賣貨便的「店到宅」冷藏服務**（190 元），由黑貓配送到府。

### 對自建 POS/電商網站的建議

1. **首選串接平台**：綠界 ECPay — 一次整合金流 + 物流（含冷凍），文件完整、SDK 多語言
2. **冷凍店到店**串接全家（99 元）+ 7-11（129 元）以給消費者選擇
3. **冷藏/冷凍宅配**串接黑貓低溫（160 元起），品質最穩定
4. 月出貨量 > 200 件時，建議直接向黑貓簽企業合約取得折扣價
5. 中長期若量大，可考慮 CYBERBIZ 的多溫層倉儲物流整合方案
